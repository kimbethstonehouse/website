Recent Work

JIT-as-a-Service

My current research focuses on JIT compilation as a service. Rather than having each device perform JIT compilation locally, we propose having specialised JIT compilation servers that devices can query for compiled code. These JIT servers can then be optimised using hardware acceleration to produce high-quality code quickly and efficiently, and they can cache frequently used applications, reducing latency and power consumption.

Multicore Processing for a Research OS

I redesigned a research operating system to support multicore processing. The research operating system is InfOS, a 64-bit x86 operating system written in C++ following an object-oriented design. By redesigning InfOS specifically with multiple cores in mind, rather than adding multicore retrospectively, we produce a well-structured operating system with a modular schedular, which is agile enough to be used to research intelligent scheduling approaches.
